À propos de StreetPhare

Navigation piétonne sécurisée · Solidarité citoyenne · Logiciel 100% libre

Notre mission

🎯 Sécuriser les déplacements piétons en milieu urbain lors d'événements de grande envergure, grâce à la solidarité citoyenne et au réseau décentralisé.

StreetPhare est une application de navigation piétonne sécurisée conçue pour renforcer la sécurité collective lors d'événements de grande envergure. L'application permet aux utilisateurs de :

🗺️

Navigation sécurisée

Calcul d'itinéraires piétons en temps réel en tenant compte des zones de danger signalées collectivement.

🔔

Alertes terrain

Signalement collaboratif des dangers (barrages, nasses, zones de rassemblement à risque) validé par consensus.

📱

Réseau P2P

Communication décentralisée sans serveur central. Aucune donnée personnelle n'est collectée.

🛡️

Anonymat absolu

Identifiants éphémères renouvelés automatiquement. Zéro traçabilité, zéro profilage.

🧭 Intégration du cœur algorithmique d'OsmAnd Core

StreetPhare utilise le cœur algorithmique d'OsmAnd Core pour le routage piéton local. Cette intégration est réalisée en respectant scrupuleusement les termes de la licence GPLv3 sous laquelle OsmAnd est distribué.


IMPORTANT : StreetPhare est un projet indépendant, non affilié, non sponsorisé, et non approuvé par OsmAnd BV. L'utilisation du code d'OsmAnd Core est conforme aux dispositions de la licence GPLv3, qui autorise explicitement la réutilisation du code sous la même licence.

OsmAnd est une application de cartographie et de navigation open-source. Pour en savoir plus, rendez-vous sur osmand.net.

La couche de routage de StreetPhare utilise le moteur de routage OsmAnd Core en tant que bibliothèque algorithmique, intégrée dans le processus de build de l'application Flutter via le répertoire OsmAnd-master/. Toute modification du code d'OsmAnd intégré dans StreetPhare est publiquement visible dans le dépôt GitHub, conformément aux exigences de la GPLv3.

Charte éthique

StreetPhare repose sur les principes suivants :

  • Sécurité des personnes : l'objectif premier est la protection physique des utilisateurs.
  • Anonymat : aucune donnée personnelle n'est collectée, stockée ou transmise.
  • Liberté : le code est 100% libre sous GPLv3. Toute personne peut l'utiliser, le modifier, le distribuer.
  • Transparence : le code source est public. Aucune manipulation cachée n'est possible.
  • Solidarité citoyenne : l'application est un outil de coopération bénévole, non commerciale.
  • Neutralité : l'application ne prend parti pour aucune cause. Elle est exclusivement un outil de navigation piétonne sécurisée et de prévention des risques physiques.
